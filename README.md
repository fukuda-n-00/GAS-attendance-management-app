# GAS-attendance-management-app

初めてGoogle App Scriptを使って、およそ3日間で勤怠管理アプリを作成しました。
改善点があればご指摘いただけますと幸いです。

## 実装した機能
- ログイン(どの企業か、管理者か従業員かを判別するため)
- 打刻
- 打刻履歴の閲覧(管理者)
- キャラクター(つばめさん)がひとことおしゃべりしてくれる

## 追加したい機能

- パスワードのハッシュ化
- 見た目やUIを整える
- ユーザー追加機能
- キャラクターとの簡単なおしゃべり機能
  - 時間帯や勤務の状態に合わせたあいさつ・おうえんをしてくれる
  - キャラクターが操作方法を案内してくれる
